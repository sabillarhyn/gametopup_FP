Tutorial Melakukan Pembelian UC PUBG




1. Masukkan User ID anda dan pilih nominal top-up yang diinginkan





2. Masukkan email anda kemudian klik tombol buat pesanan





3. Jika tombol berubah menjadi pesanan masuk berarti pesanan sudah berhasil dibuat







Nikmati pengalaman pembelian tanpa repot untuk membeli kredit game di NPC Store, platform top-up untuk game kegemaran anda.
